# Swasth Hospital Website

A responsive healthcare website designed to showcase hospital services, facilities, and information with a clean and modern UI.

Live Demo: https://swasth-hospital.vercel.app/

## ScreenShots

## Tech Stack

- Next.js
- React
- Tailwind CSS
- GSAP
- Lenis
- Vercel

## Features

- Modern responsive UI
- Smooth scrolling and animations
- Clean layout for healthcare information
- Reusable React components
- Fast page load performance

## Project Overview

This project demonstrates building a **production-ready healthcare landing website** using modern frontend technologies. It focuses on user experience, smooth animations, and responsive design.
